# SearchFIT SEO Toolkit

Free AI-powered SEO toolkit by [SearchFit.ai](https://searchfit.ai) — works with Claude Code, and any AI agent that supports Claude Code plugins.

Audit websites, plan content strategy, optimize pages, generate schema markup, cluster keywords, track AI visibility, and more.

## Installation

Install from the Claude Code marketplace:

```bash
/plugin install searchfit-seo
```

Or install directly from GitHub:

```bash
/plugin marketplace add searchfit/searchfit-seo
/plugin install searchfit-seo
```

## What's Included

### Skills (11)

Skills activate automatically when you ask about these topics:

| Skill                   | What It Does                                            |
| ----------------------- | ------------------------------------------------------- |
| **SEO Audit**           | Comprehensive website SEO health check                  |
| **Technical SEO**       | Core Web Vitals, crawlability, indexation, security     |
| **On-Page SEO**         | Optimize individual pages for target keywords           |
| **Broken Links**        | Find and fix dead links hurting your SEO                |
| **Internal Linking**    | Improve site architecture and link equity flow          |
| **Schema Markup**       | Generate JSON-LD structured data for rich results       |
| **Content Strategy**    | Plan a data-driven content roadmap                      |
| **Content Brief**       | Create detailed briefs for writers (human or AI)        |
| **Keyword Clustering**  | Group keywords into topical clusters                    |
| **AI Visibility**       | Track and improve your brand's presence in AI responses |
| **Content Translation** | Translate and localize content for international SEO    |

### Agents (3)

Autonomous agents that handle complex, multi-step tasks:

| Agent                   | What It Does                                           |
| ----------------------- | ------------------------------------------------------ |
| **SEO Auditor**         | Full autonomous site audit with scored report          |
| **Content Strategist**  | Analyzes your site and creates a complete content plan |
| **Competitor Analyzer** | Researches competitors and identifies opportunities    |

### Commands (6)

Quick actions you can invoke directly:

| Command                         | What It Does                            |
| ------------------------------- | --------------------------------------- |
| `/create-topic <seed>`          | Research and generate a full topic plan |
| `/create-content <topic>`       | Generate a full SEO-optimized article   |
| `/translate-content <language>` | Translate content with multilingual SEO |
| `/seo-check [file]`             | Quick SEO check on a page               |
| `/generate-schema [type]`       | Generate JSON-LD structured data        |
| `/keyword-cluster <keywords>`   | Cluster keywords into content groups    |

## Examples

```
# Research topics for your niche
/create-topic "SaaS marketing"

# Run a full SEO audit
"Audit the SEO of my website"

# Create content
/create-content "how to improve website speed"

# Quick check on a file
/seo-check src/app/page.tsx

# Translate to Spanish for Mexico
/translate-content Spanish --market Mexico

# Cluster your keywords
/keyword-cluster "seo tools, best seo software, seo audit, website optimization, keyword research tool"

# Generate schema for a page
/generate-schema article --file src/app/blog/[slug]/page.tsx

# Analyze competitors
"Compare my SEO against competitor1.com and competitor2.com"
```
